[문제상황]  
그레고리력의 정확한 윤년 규칙은 다음과 같다. 4로 나누어 떨어지면서 100으로 떨어지지 않으면  
윤년이다. 400으로 나누어 떨어지는 것도 윤년에 포함된다. 년도를 입력받아 그것이 윤년이면  
True/Flase를 출력하는 프로그램을 작성하시오.  
```python
y=int(input('년도를 입력하시오: '))
print(y%4==0 and not(y%100==0) or y%400==0)
```
[출력결과]  
```python
년도를 입력하시오: 2020  
True
```
```python
년도를 입력하시오: 2021  
False
```
